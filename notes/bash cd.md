# bash cd

[[bash dirs]], [[bash pushd]], [[bash popd]], [[bash pwd]]

```sh
cd      # move to homedir
cd ~    # move to ~ expands to homedir `$HOME`

cd -    # change to previous dir

```
## shopt

[[bash shopt]]
```sh
shopt -s cdspell    # correct misspelled dir
```
