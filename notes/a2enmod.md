# a2enmod
> enable apche2 modules

## usage
```sh
a2enmode ssl                    # enable-modules

a2enmod info.load               # /server-status

a2enmod info.conf  
# vim /etc/apach2/mods-available/info.conf => comment Require to access outside localhost
```
