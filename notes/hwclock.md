# hwclock

> hwclock - query and set the hardware clock (RTC) 

## usage
```sh
hwclock

hwclock --systohc --debug   # what it does when we copy system time to hardware time.

hwclock -hctosys --debug    # what it does when we copy hardware time to system time.
```
