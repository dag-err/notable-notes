# ruby rvm

```sh
source ~/.rvm/bin/rvm

rvm list known       # lists installable ruby versions
rvm list rubies     # list installed rubies

rvm install 2.0.0

bash -l -c "rvm use 2.0.0"    # http://stackoverflow.com/a/30739863
```
