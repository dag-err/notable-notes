# scala repl

`read evaluate print loop`

## start repl
```sh
scala     # start repl
```
## commands
```sh
scala> :help

scala> :reset  // clear past commands in session

scala> :type   // print type without evaluating
```

## repl
```
scala> val x, y = 1
```
