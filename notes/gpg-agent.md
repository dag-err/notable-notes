# gpg-agent


## gpg-connect-agent

> Connect to a running agent and send commands
> `gpg-connect-agent` runs via `pinentry` interactively or by passing options.
```sh
gpg-connect-agent reloadagent /bye      # reload agent on osx to reenter passphrase for gpg
```
